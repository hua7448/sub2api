# sub2api-site

SmartQ 客户使用文档站点，基于 Next.js 构建。

## Getting Started

```bash
pnpm install
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser.

## Build

```bash
pnpm build
```

静态产物输出到 `out/` 目录。

## Deploy

```bash
cd deploy
./start.sh
```

## Configuration

编辑 `lib/docs-data.ts` 修改站点信息、价格、FAQ 等。
